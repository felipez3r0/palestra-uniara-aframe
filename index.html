WebVR

WebVR é uma plataforma de realidade virtual com todas as vantagens da web: aberta, conectada e instantânea.

A-frame

A-Frame é um framework opensource, lançado em dezembro de 2015 pela Mozilla para a criação de cenas 3D e de realidade virtual para a Web.

Instalando

          
https://aframe.io/releases/0.8.0/aframe.min.js

$ npm install aframe

Código do exemplo

          
<a-scene>
  <a-entity rotation="0 45 45" geometry="primitive: box"
    position="0 2 -2" material="src: img/resources/firefox.jpg"></a-entity>
  <a-sky color="#222"></a-sky>
</a-scene>
        
      
<html>
  <head>
    <script src="js/aframe.min.js"></script>
  </head>
  <body>
    <a-scene>
      <a-entity rotation="0 45 45" geometry="primitive: box"
        position="0 2 -2" material="src: img/resources/firefox.jpg">
      </a-entity>
      <a-sky color="#222"></a-sky>
    </a-scene>
  </body>
</html>
